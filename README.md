# Portfolio Optimization (Efficient Frontier)

This project implements Modern Portfolio Theory (MPT) to construct the efficient frontier using historical stock returns of major US tech companies: AAPL, MSFT, GOOG, and AMZN.

---

## 📊 Project Overview

- Calculate daily returns from historical adjusted close prices
- Estimate the covariance matrix of returns
- Generate 5,000 random portfolios with different weights
- Compute portfolio return, volatility, and Sharpe ratio
- Visualize the efficient frontier with color-coded Sharpe ratio

---

## 🛠 Tools & Libraries

- Python  
- Pandas for data manipulation  
- NumPy for numerical computation  
- Matplotlib for visualization  
- yfinance for historical market data

---

## 💾 Dataset

- Stock prices (AAPL, MSFT, GOOG, AMZN) from 2018-01-01 to 2024-01-01
